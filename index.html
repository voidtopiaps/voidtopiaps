⚙️ How to Play

🟢 Android / Power Tunnel

1 POWER TUNNEL SETUP
VoidTopia Link: https://growsoft.cc/android.txt
2 App / APK Installation

Download and install Power Tunnel from a trusted source. For detailed installation, click the Discord support button above.

3 Configuration

Import the VoidTopia link into Power Tunnel and enable the connection. Launch Growtopia and connect to Server Name: Voidtopiaps.

🍎 iOS

1 Download App called: "Surge 5" from App Store. Search for Surge 5 iOS.
2 Press "OK" to grant necessary permissions.
3 Click on "Default.conf" in the app menu.
4 Click on "IMPORT" part - Download Profile from URL.
5
Put URL from your copy when click iOS host button above and click "OK" and "Done".
6 Press "SETUP" and then agree to policy by clicking "OK" and "Allow" for VPN Configuration!
7 Done! Open "Growtopia" and connect to Server Name: Voidtopiaps.
8 For iOS Link & Support: Click the Discord support button above for detailed guidance.

💻 Mac

1 Press ⊞ Win+R (Windows Key + R) to open up the "Run" window.
2 Type in C:\Windows\System32\drivers\etc in the textbox.
3 Find file named hosts.
4 Right-click on "hosts" file and edit it with Notepad/text editor.
5 Add these IPs to the hosts file: 
Server Name: Voidtopiaps
91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com
91.134.85.13 www.growtopia2.com
6 Save the file and restart your computer if necessary.
7 Launch Growtopia and log on to Server Name: Voidtopiaps.

🖥️ Computer (Windows)

1 Press ⊞ Win+R (Windows Key + R) to open up the "Run" window.
2 Type in C:\Windows\System32\drivers\etc in the textbox.
3 Find file named hosts.
4 Right-click on "hosts" file and edit it with Notepad/text editor (Run as Administrator).
5 Add these IPs to the hosts file: 
Server Name: Voidtopiaps
91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com
91.134.85.13 www.growtopia2.com
6 Save the file and flush DNS cache by running ipconfig /flushdns in Command Prompt.
7 Launch Growtopia and log on to Server Name: Voidtopiaps.
