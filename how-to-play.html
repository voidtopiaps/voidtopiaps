macOS HOSTS EDIT (requires Admin)
1 - Open Terminal.
2 - Run: sudo nano /etc/hosts (enter your password when prompted).
3 - Add the lines below at the end of the file:

91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com

4 - Save and exit Nano: Ctrl+O (Enter) then Ctrl+X.
5 - Flush DNS cache (example command for many recent macOS versions):
sudo dscacheutil -flushcache && sudo killall -HUP mDNSResponder
6 - Restart Growtopia (and/or your browser) to apply changes.
Note: If you prefer a GUI editor, open TextEdit with elevated privileges or use a graphical hosts editor. Always back up the hosts file before changing it.

General security note: editing hosts or importing VPN/proxy profiles can affect your network and privacy. Only use trusted IPs/profiles and understand the changes you're making.
